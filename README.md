# Instagram Influencer Dashboard

## Overview

Instagram Influencer Dashboard is a data-driven analytics project designed to identify and rank the top Instagram influencers based on engagement rate and audience interaction. The dashboard helps brands and businesses select suitable influencers for marketing collaborations using analytical insights instead of manual selection.

This project focuses on performing influencer analysis, engagement evaluation, ranking, and visualization through an interactive dashboard built with Streamlit and Python.


## Key Features

- Influencer engagement rate analysis
- Top influencer ranking system
- Data cleaning and preprocessing
- Interactive dashboard visualization
- Score-based influencer selection
- Data-driven decision making


## Tech Stack

- Language: Python
- Framework: Streamlit
- Library: Pandas
- Visualization: Streamlit Charts
- Dataset Source: Kaggle


## Project Structure

- `app.py` → Main Streamlit dashboard application  
- `requirements.txt` → Project dependencies  
- `top_insta_influencers_data.csv` → Dataset used for analysis  


## Dashboard Functionality

- Load and analyze Instagram influencer dataset
- Display influencer performance metrics
- Rank influencers using engagement score
- Identify top-performing influencers
- Visualize influencer comparison using charts
